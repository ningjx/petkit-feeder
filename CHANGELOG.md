# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-05

### Added
- Initial release
- PetKit feeder integration for Home Assistant
- Support for Fresh Element Solo (D4)
- Feeding schedule management
- Feeding history tracking
- Manual feeding control
- Device status monitoring (online, WiFi, desiccant)
- Dedicated Lovelace card

### Features
- **Feeding Schedule Management** - Add/edit/delete feeding schedules with weekly sync
- **Feeding History** - Track every feeding detail
- **Manual Feeding** - One-click feeding
- **Status Monitoring** - Online status, WiFi signal, desiccant status
- **Lovelace Card** - Custom visualization card

### Supported Devices
- Fresh Element Solo (D4) ✅
- Fresh Element (D3) 🚧
- Fresh Element Duo (D4s) 🚧
- Feeder Mini (Mini) 🚧
